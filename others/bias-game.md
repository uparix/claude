# BIAS GAME

## Game Concept

Build 2 groups in the room. The group should not be larger than 1-4 people. The first group gets these 3 words:

- BAT
- LEMON
- CINERAMA

The second group gets these 3 words:

- WHIRL
- SLAPSTICK
- CINERAMA

Give a limited amount of time (usually 2-3 Minutes) to solve the problem.

## Conclusion

The correct solution for group 1 could be:

- TAB
- MELON
- AMERICAN

_The second group usually cannot come up with a solution for the first 2 words.
The third word would be possible but they don't get it, because they could not solve the first 2 words._
